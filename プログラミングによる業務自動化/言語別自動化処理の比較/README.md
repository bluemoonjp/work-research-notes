# 言語別自動化処理の比較

業務で使う自動化処理を複数言語で実装し、コードの書き方・特徴・効率を比較する。

## ディレクトリ構成

処理単位ごとにディレクトリを作成する。

```
言語別自動化処理の比較/
  <処理名>/
    概要.md
    <処理名>_CSHARP.cs
    <処理名>_VBA.bas
    言語別解説.md
```

## 各ファイルの役割

| ファイル | 内容 |
|---|---|
| `概要.md` | 処理の目的・前提条件・想定シナリオを説明する。コードは含めない。 |
| `<処理名>_CSHARP.cs` | C# による実装サンプル。そのままビルドできる完成度とする。 |
| `<処理名>_VBA.bas` | VBA による実装サンプル。そのまま VBE に貼り付けて実行できる完成度とする。 |
| `言語別解説.md` | 各言語の特徴・効率的に書ける部分・非効率になりがちな部分を言語横断で解説する。 |

## 言語の追加ルール

- デフォルト言語は C#（`.cs`）と VBA（`.bas`）
- 言語を追加する場合はファイルを追加するだけでよい（既存ファイルの変更不要）
- ファイル名のサフィックスは言語名の大文字略称とする（例：`_PYTHON.py`、`_POWERSHELL.ps1`）
- `言語別解説.md` に追加言語の節を加える

## 調査・作成規約

CLAUDE.md の調査規約に従い、実装サンプルは Codex と Claude サブエージェントで並行調査する。
