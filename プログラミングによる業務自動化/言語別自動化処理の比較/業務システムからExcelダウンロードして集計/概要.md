# 業務システムからExcelダウンロードして集計

## 処理の目的

ログインが必要な社内 Web システムにブラウザでアクセスし、売上データ（Excel）をダウンロードする。
ダウンロードしたデータを別の Excel ファイルに転記し、カテゴリ別の集計シートを作成する。

## 想定シナリオ

- 社内 Web システムの売上レポート画面にログインする
- 「エクスポート」ボタンをクリックして Excel（.xlsx）をダウンロードする
- ダウンロードした Excel の中身を読み込み、カテゴリ別に集計して新しい Excel に出力する

コード内の URL・セレクタ・ログイン情報はすべてダミー。実環境では定数を差し替える。

## 前提条件

### NuGet パッケージ

| パッケージ | 用途 | ライセンス |
|---|---|---|
| `Microsoft.Playwright` | ブラウザ操作 | Apache 2.0（商用無料） |
| `ClosedXML` | Excel 読み書き | MIT（商用無料） |

### 初回セットアップ

NuGet パッケージをインストール後、Chromium ブラウザのバイナリを取得する（初回のみ）:

```
dotnet tool install --global Microsoft.Playwright.CLI
playwright install chromium
```

## 入力 / 出力

| 区分 | 内容 |
|---|---|
| 入力 | コード内の定数（ログイン URL・ID・パスワード） |
| ダウンロード | 業務システムが出力する売上 Excel（.xlsx） |
| 出力 | `集計結果.xlsx`（生データシート＋カテゴリ集計シート） |

## ダウンロード Excel の想定フォーマット

| 列 | 内容 | 型 |
|---|---|---|
| A | 注文日 | 文字列（例: 2024-06-01） |
| B | 商品名 | 文字列 |
| C | カテゴリ | 文字列 |
| D | 単価 | 数値 |
| E | 数量 | 整数 |

1行目はヘッダ行、2行目以降がデータ行。
