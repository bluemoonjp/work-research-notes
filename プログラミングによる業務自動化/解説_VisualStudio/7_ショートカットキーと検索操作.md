# 7: ショートカットキーと検索操作

## この記事で学ぶこと

- ビルド・実行系のショートカット
- コード移動・検索系のショートカット
- ソリューション全体の検索・置換
- リファクタリング系の操作
- ショートカットのカスタマイズ方法

---

## ビルド・実行系

| キー | 操作 |
|------|------|
| F5 | デバッグ開始 |
| Ctrl+F5 | デバッグなしで実行 |
| Shift+F5 | デバッグ停止 |
| F6 | ビルド |
| Ctrl+Shift+B | ソリューション全体をビルド |
| F10 | ステップ オーバー（デバッグ中） |
| F11 | ステップ イン（デバッグ中） |
| F9 | ブレークポイントの追加/削除 |

---

## コード移動系

| キー | 操作 |
|------|------|
| F12 | 定義へ移動（メソッド・クラスの宣言箇所に飛ぶ） |
| Alt+F12 | 定義をピーク（別ウィンドウで定義を確認） |
| Shift+F12 | すべての参照を検索（どこで使われているか一覧表示） |
| Ctrl+- | 前の位置に戻る |
| Ctrl+Shift+- | 次の位置に進む |
| Ctrl+G | 行番号を指定してジャンプ |
| Ctrl+T | 型・メンバー名で検索してジャンプ（Go to All） |

**F12** と **Shift+F12** の組み合わせはコードを読む際に非常に役立つ。定義を確認 → 参照先を確認 → 戻る、の流れで追いかけられる。

---

## 検索・置換

| キー | 操作 |
|------|------|
| Ctrl+F | 現在のファイル内を検索 |
| Ctrl+H | 現在のファイル内を置換 |
| Ctrl+Shift+F | ソリューション全体を検索 |
| Ctrl+Shift+H | ソリューション全体を置換 |

### ソリューション全体の検索（Ctrl+Shift+F）

検索対象を「ソリューション全体」に切り替えると、すべての `.cs` ファイルを横断して検索できる。変数名・メソッド名・文字列リテラルなどを一括検索するときに使う。

正規表現検索も使用可能（検索ボックス右の `.*` ボタンで有効化）。

---

## リファクタリング系

| キー | 操作 |
|------|------|
| F2 | シンボルのリネーム（変数・メソッド・クラス名を一括変更） |
| Ctrl+. | クイック アクションと修正候補（電球アイコン）を表示 |
| Ctrl+R, Ctrl+M | メソッドの抽出（選択範囲を新しいメソッドに切り出す） |
| Ctrl+K, Ctrl+D | ドキュメント全体をフォーマット |
| Ctrl+K, Ctrl+F | 選択範囲をフォーマット |

**Ctrl+.** は最もよく使うキーの一つ。未解決の参照に `using` を追加する、エラーを自動修正する、インターフェイスのスタブを生成するなど、状況に応じた修正候補を提示してくれる。

**F2 のリネーム**は同じシンボル名をすべてのファイルで一括変更するため、手動の置換より安全。コメント内の文字列は変更されないので注意。

---

## コメントアウト

| キー | 操作 |
|------|------|
| Ctrl+K, Ctrl+C | 選択行をコメントアウト |
| Ctrl+K, Ctrl+U | 選択行のコメントを解除 |

---

## ショートカットのカスタマイズ

**ツール > オプション > 環境 > キーボード**

コマンド名で検索してキーを割り当てられる。よく使うコマンドを覚えやすいキーに変更しておくと便利。

---

## コマンド検索（Ctrl+Q）

ショートカットを覚えていなくても、**Ctrl+Q**（クイック起動）でコマンド名を入力すれば操作できる。「NuGet」「デバッグ」「フォーマット」など、日本語でも検索できる。
