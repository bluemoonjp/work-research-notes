# 解説_VisualStudio

## このシリーズの目的

Visual Studio を使った C# 業務自動化開発を想定し、IDE 自体の設定・使い方を体系的に解説する。言語仕様の解説は行わず、「Visual Studio をどう使うか」に絞った記事群。

## 対象読者

Visual Studio で C# プログラミングを始める人。特に業務自動化ツールを作りたいと考えている人を想定する。他のエディタや IDE の経験があってもよい。
