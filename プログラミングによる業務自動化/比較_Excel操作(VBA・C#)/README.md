# Excel操作を比較する

## 処理の目的

商品マスタと注文の 2 シートを持つ Excel ファイルを操作する 4 つのプログラムを、VBA と C# で実装し比較する。

| # | プログラム名 | 処理の概要 |
|---|---|---|
| 1 | 単純なWhere条件 | 商品マスタをフィルタして新シートに書き出す |
| 2 | ジョイントグループ化 | 2 シートを結合し、年月・商品コードで集計する |
| 3 | 複雑なWhere条件 | OR を含む複合条件でフィルタして新シートに書き出す |
| 4 | 1〜3 まとめ | プログラム 1〜3 を 1 本にまとめて連続実行する |

## 想定シナリオ

- 同じ Excel ファイル内に「商品マスタ」「注文」の 2 シートが存在する
- 処理結果は同じファイル内の新しいシートに書き出す（既存シートは上書き）
- プログラムは Excel を開いた状態（VBA）または外部から Excel ファイルを操作（C#）として動作する

## 前提条件

### VBA

特別なライブラリ追加は不要。「Scripting.Dictionary」は標準で利用可能。

VBE（Visual Basic Editor）に 4 つのモジュールをインポートして使用する。

### C#

| NuGet パッケージ | 用途 | ライセンス |
|---|---|---|
| `ClosedXML` | Excel 読み書き | MIT（商用無料） |

Excel ファイルは実行ファイルと同じディレクトリに `データ.xlsx` として配置する。

## 入力データのフォーマット

### 商品マスタ（1 行目ヘッダ）

| 列 | 項目 | 型 | 備考 |
|---|---|---|---|
| A | 商品コード | 文字列 | 注文シートとの結合キー |
| B | 商品名 | 文字列 | |
| C | 型番 | 文字列 | 末尾が "HG" でハイグレードモデルと判定 |
| D | 発売日 | 日付 | 今年かどうかの判定に使用 |
| E | 仕入元 | 文字列 | "A社" / "B社" 等 |
| F | 調達区分 | 文字列 | "国内調達" / "受注生産" 等 |
| G | 単品原価 | 数値 | |
| H | 単品売価 | 数値 | |

### 注文（1 行目ヘッダ）

| 列 | 項目 | 型 | 備考 |
|---|---|---|---|
| A | 注文番号 | 文字列 | |
| B | 注文日 | 日付 | プログラム 2 で販売年月（yyyy/mm）の算出に使用 |
| C | 商品コード | 文字列 | 商品マスタとの結合キー |
| D | 数量 | 整数 | |

## 出力シートの仕様

### プログラム 1: 最新ハイグレードモデル一覧

商品マスタの全列（A〜H）をそのまま書き出す。

フィルタ条件:
- 発売日が今年
- 型番の末尾が "HG"

### プログラム 2: 販売集計

| 列 | 項目 | 算出方法 |
|---|---|---|
| 1 | 販売年月 | 注文日を yyyy/mm 形式に変換 |
| 2 | 商品コード | 結合キー |
| 3 | 商品名 | 商品マスタより |
| 4 | 単品原価 | 商品マスタより |
| 5 | 単品売価 | 商品マスタより |
| 6 | 売上個数 | SUM(数量) |
| 7 | 売上金額 | SUM(数量 × 単品売価) |
| 8 | 原価 | SUM(数量 × 単品原価) |
| 9 | 利益額 | 売上金額 − 原価 |
| 10 | 利益率 | 利益額 ÷ 売上金額（% 表示） |

ソート順: 販売年月 昇順 → 利益額 降順

### プログラム 3: イレギュラー発注商品

商品マスタの全列（A〜H）をそのまま書き出す。

フィルタ条件:
- （仕入元 = "A社" かつ 調達区分 = "国内調達"）または（仕入元 = "B社" かつ 調達区分 = "受注生産"）
