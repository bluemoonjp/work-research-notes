# C# 言語解説シリーズ

## このシリーズの目的

プログラミング経験者（他言語ユーザー・C# 初学者）が、C# の文法・書き方を体系的に確認するための記事群です。

Java・Python・VBA などの経験者が「C# ではどう書くか」「他言語と何が違うか」を効率よく把握できるよう、各記事は C# 固有の書き方と他言語との違いを中心に構成しています。

## 対象読者

- 他言語（Java・Python・VBA など）の経験があり、C# を新たに学ぶ方
- C# を使い始めたが、言語仕様を体系的に整理したい方
- モダンな C#（C# 8 以降）の機能を把握したい方
