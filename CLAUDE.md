# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## リポジトリ概要

仕事で使える技術的な調査結果を記録するナレッジベース。AI が調査した内容を蓄積することを主目的とする。

## コンテンツルール

- **1ページ1テーマ**: 各 Markdown ファイルは単一のテーマ・トピックを扱う
- **出典リンク必須**: 主要な情報源は必ずリンクを残す
- **組織情報の排除**: 所属組織の固有名詞・内部情報はリポジトリに含めない

## ファイル・ディレクトリ命名規則

- README.md・CLAUDE.md などの標準ファイルを除き、ファイル名・ディレクトリ名は日本語を使用する
- コードサンプルは記事と同じディレクトリに置き、ファイル名で言語を区別する（例: `ブラウザ操作_CSHARP.cs`）

## Git 規約

- ファイルに変更を加えたら、都度 `git commit` する

## 言語規約

- **基本想定言語**: C# および VBA
- コードサンプルを作成する際は、特に指定がない限り C# または VBA を使用する

## ディレクトリ構成パターン

大テーマ → 中テーマ（任意） → 小テーマ（任意） → 記事・コードサンプル

```
<大テーマ>/
  <記事>.md
  <中テーマ>/
    <記事>.md
    <小テーマ>/
      <記事>.md
      <コードサンプル>.<拡張子>
```
